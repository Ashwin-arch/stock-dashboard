
# Stock Dashboard

## Overview

The Stock Dashboard is a web-based application designed to track and monitor real-time and historical stock prices. It provides users with a clean and interactive interface to view price changes of multiple stocks and understand market trends.

## Features

* View real-time stock prices of selected companies
* Track multiple stocks simultaneously
* Display key price metrics such as open, high, low, and close
* Interactive charts for price visualization
* Search and add stock symbols dynamically
* Responsive layout for desktop and mobile devices

## Tech Stack

* Frontend: HTML, CSS, JavaScript
* Data Source: Web-based stock data services (no API keys required)
* Tools: Browser-based web technologies only

## Project Structure

```
stock-dashboard/
│
├── index.html        # Main dashboard interface
├── style.css         # Styling and layout
├── script.js         # Stock data fetching and logic
├── assets/           # Images and icons
└── README.md         # Documentation
```

## Working of the Dashboard

1. User selects or enters stock names or symbols.
2. The application fetches stock data using web tools such as embedded services or publicly accessible endpoints.
3. Stock prices and changes are displayed dynamically on the dashboard.
4. Charts update automatically when data changes.
5. All processing happens on the client side without any backend or API authentication.

## Installation and Setup

1. Clone or download the project:

```
git clone https://github.com/Ashwin-arch/stock-dashboard.git
```

2. Open the project folder.
3. Launch `index.html` directly in any modern web browser.

No server, backend, or installation of dependencies is required.

## Data Handling

This project does not use API keys or backend services.

Stock information is accessed using web tools and publicly available data sources, making the dashboard simple, lightweight, and easy to deploy.
